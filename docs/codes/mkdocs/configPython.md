
# 配置Python环境

## 环境
> - Windows 10
> - anaconda 4.8.2
> - Python 3.7.6

1. 直接安装 `anaconda` 虚拟环境 [anaconda主页](https://anaconda.org/)
2. 选择自己的电脑系统进行安装~
3. 直接下一步下一步安装到电脑上~  


## 安装Mkdocs

> 直接通过的 `Python` 的 `pip` 包管理进行安装
```
pip install mkdocs
```

> 主要使用 `material` 主题

- 安装不同页面样式的主题

```
pip install mkdocs mkdocs-material
```

- or 直接下载主题到本地直接配置和更改

```
git clone https://github.com/squidfunk/mkdocs-material.git
```
