# 使用Mkdocs集成博客

- [Mkdocs主页](https://www.mkdocs.org).

## 命令

* `mkdocs new [dir-name]` - 创建一个项目.
* `mkdocs serve` - 进入创建的文件夹开启服务.
* `mkdocs build` - 建立文档站点.
* `mkdocs -h` - 打印帮助.

## 项目布局

    mkdocs.yml    # 配置文件
    docs/
        index.md  # 主页.
        ...       # 其他markdown页面, 图片已经其他文件.
